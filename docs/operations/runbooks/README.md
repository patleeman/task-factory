# Runbooks

Store repeatable operational procedures in this directory.

## Naming convention

`service-or-scope-action.md` (example: `api-rollback-release.md`)

## Authoring steps

1. Copy [`Runbook template`](../../templates/RUNBOOK_TEMPLATE.md)
2. Fill prerequisites, procedure, validation, and rollback
3. Add owner/on-call team and escalation contacts
